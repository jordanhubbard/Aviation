# Deployment Guide

> **Production Deployment Guide for Aviation Monorepo Applications**

This comprehensive guide covers deploying all aviation applications to production environments including Docker, Kubernetes, and cloud platforms.

## Quick Start

```bash
# Clone repository
git clone https://github.com/jordanhubbard/Aviation.git
cd Aviation

# Build and deploy with Docker Compose
docker-compose up -d

# Check status
docker-compose ps
```
